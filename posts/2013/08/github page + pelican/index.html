What is Pelican?

Pelican是一個靜態博客生成器，具有常見博客系統如WordPress的基本功能，在功能、速度、開放性上更符合Geek們的口味。

靜態博客的優點：

  • No-SQL。原始文章再也不是數據庫表中的一個值，而是實實在在可以用任意編輯器編輯的純文本文件，撰寫、修改文章都不再依賴專門工具。
  • 源文件支持多種標記語言（與轉換引擎相關），簡化文章編輯。
  • 速度快。不再需要查詢數據庫，直接讀取靜態html，因此速度飛快。
  • 本地（同步）備份網站，網站遷移部署更靈活。

靜態博客的缺點：

  • 評論依賴外部服務。依賴第三方評論系統。
  • 修改設置，修改、發表文章大部分情況下需要編譯整個網站。
  • 通常依賴命令行工具，具有一定技術門檻。

Pelican的功能與特點：

  • 基於python環境，如Jinja2、pygments
  • 標記語言支持：reStructuredText、Markdown、AsciiDoc，支持自定義
  • 命令行操作
  • 主題/插件自定義
  • 文章可發佈多種語言版本
  • Atom/RSS訂閱
  • 多種常用第三方服務集成：Github、Disqus、Google Analytics、Twitter

Install Pelican(windows)

我是在windows底下測試，使用 pip 安裝package

pip install pelican
pip install Markdown

Start Pelican

首先創建一個博客目錄用於存放所有的文件,然後執行 pelican-quickstart 回答幾個基本的問題，執行完畢在目錄下就生成了基礎的文件和目錄如下：

  • content/，默認posts文檔（*.rst *.md *.html等等）存放目錄
  • output/，默認靜態網站編譯輸出目錄，把該目錄下文件拷貝到任意開啟HTTPServer的目錄下並綁定好設置的域名即可訪問。
  • Makefile，pelican使用make進行基本操作，請仔細閱讀該文件，瞭解常用的make選項。
  • pelicanconf.py，pelican配置文件
  • publishconf.py，發佈博客的配置文件，由make調用，可以自行設置
  • develop_server.sh，測試pelican的腳本，由make調用，一般不需修改

常用的make選項：

  • make html，用pelican生成靜態網頁
  • make remove，刪除輸出目錄的文件
  • make serve，用 python -m SimpleHTTPServer 在輸出目錄運行HTTP服務，可以從 localhost:8000 訪問。
  • make regenerate，自動檢測文件更新並重新生成html
  • make devserver，運行HTTP服務並檢測文件更新，一旦更新就重新生成html
  • make publish，發佈靜態網站
  • make github，發佈到 gh-pages

Code highlight

pelican使用pygments來處理代碼高亮

  • reST格式高亮

reST格式可以採用`code-block`原語，編輯時注意前後空行以及縮進與必要的空格（python在這方面很嚴格）

.. code-block:: identifier

   <indented code block goes here>

輸出的html代碼為：

<div class="highlight"><pre><span class="p">..</span> <span class="ow">code-block</span><span class="p">::</span> identifier

   &lt;indented code block goes here&gt;
</pre></div>

注意這裡CSS類是`highlight`。

  • Markdown高亮

Markdown需要經過python-markdown處理，並且需要codehilite擴展的支持，示例如下：

方案一：

Some text.

    :::python
    def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

輸出的html代碼為：

<p>Some text.</p>
<div class="codehilite"><pre>
</pre></div>

方案二：

Some text.

```python
def factorial(n):
if n == 0:
    return 1
else:
    return n * factorial(n - 1)
```

兩種方案輸出的CSS類都是 codehilite 而不是 highlight，另外第一種方案必須縮進，第二種則不必。

use theme

下載網友分享的自定義主題：

$ git clone https://github.com/getpelican/pelican-themes BLOGDIR/themes

在 pelicanconf.py 中添加 THEME = "./themes/THEMENAME"

dev server

移至你的pelican project裡的 output 並執行cmd 使其可藉由 localhost:8000 端口開啟

python -m http.server
